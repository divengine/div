# Release v4.0.0
Date: 2013-05-27

## Description
This release includes:
- Fixed some bugs in locations and conditional parts.
- Created a translator of dialects. Div now have 2 new public methods.
- New feature: template properties. Now you can specify some properties in the template's code, for example, the dialect of the current template.
- New feature: predefined subparsers. Div provide pre-defined sub-parsers, for example, This means that a new instance of div will be created, similar to the loops and the capsules. Other predefined subparsers will be developed in future releases.
- New feature: sub-parser's events. Now in the templates's code you can specify when a sub-parser will be executed: beforeParse, afterInclude or afterParse. Example.
- Improved the conditional parts detection.
- Changed to private some div's properties.

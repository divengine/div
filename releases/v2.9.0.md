Aug 20, 2012

- Delete the DIV_CLASS_NAME constant: now is more simple to change the name of div class. Simply change the name of div class, no more!

- Fix problem of template vars's scope. The inheritance mechanism is more simple now:

parent.tpl

```div
<!--{ define a block }-->
{= block1: 

```

...some code here...

=}

```div
<!--{ show the block }-->
{$block1}

```

child.tpl

```div
<!--{ re-define the block1 (note that the block1 var is protected) }-->
{= *block1:

```

...some another code here...

=}

```div
<!--{ extends }-->
{% parent %}

```

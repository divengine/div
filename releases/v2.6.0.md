```div
<!--{ Next, define the block that the replacement will be take effect }-->

{:replac}

```

... some code here ....

```div
{:/replac}

```

Example:

```php
{= php-code: [
        ['echo ', '<b>echo</b> '],
        ['/\'([^\'](?:\\.|[^\\\']*)*)\'/i', '<span class="string">\'$1\'</span>',true]
    ] =}

{:php-code}

<?php
echo 'hello world';
?>

{:/php-code}

```

Output:

```php
<?php
<b>echo</b> <span class="string">'hello world'</span>
?>
```

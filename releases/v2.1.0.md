# Release v2.1.0

Date: 2012-05-08

## Description

May 08, 2012

- Added a new feature: pre-processed parts.

Now you can pre-processed by div any part in template. The pre-processing are similar to include, but the pre-processing parse the code before including it.

Include is: {% part.tpl %}  (include and then parse) Pre-processing is {%% part.tpl %%} (parse and then include)

IMPORTANT!: The pre-processing have a priority with regard to the list iterations.

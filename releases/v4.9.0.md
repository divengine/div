# Release v4.9.0
Date: 2016-12-22

## Description
- add new default subparser join

Syntax:
```
{join} varname | delimiter {/join}
```

index.tpl
```
{= tags: ['a','b', 'c'] =}
{join} tags |, {/join}
{join} tags |,{/join}
{join} tags {/join}
```

Output:
```
a, b, c
a,b,c
abc
```

- important bugfix/improvement: access to parent loop

```
[$parentloop] parent =>
	[$childloop] child =>
		Parent key: {$parent._key}
		Child key: {$_key} or {$child._key}
	[/$childloop]
[/$parentloop]
```

## Description

February 19, 2013

- The documentation was updated
- Improved detection of infinite loops on includes and replacements
- Release the 3.4 version

February 17, 2013

- Add new feature: Multiple variable's modifiers

Syntax:

```div
{$varname|modifier1|modifier2|modifier3|...|}

```

index.tpl

```div
{= word: "ABCDEFG" =}

{$word|0,3|}
{$word|0,3|_|}
{$word|0,3|_|^|}
   {$word|0,3|_|^|~2|}

```

Output

```div
ABC
abc
Abc
Ab

```

About this documentation
This book documents the Div PHP Template Engine: its architecture, syntax, and execution model. Div is a deterministic, model-driven system for generating text outputs, from HTML to source code and structured data. The documentation emphasizes repeatable pipelines, dialect interoperability, and multi-step transformations built on fixed-point parsing.
  • Conceptual overview and execution model
  • Template language features with examples
  • PHP integration and engine API
  • Mechanisms: recursion, components, and inheritance
  • Appendixes and reference material

## 2016-11-16
### variable modifier for trim values
###### in the present:
         
```
{strip}{$varname}{/strip}
(# trim("{$varname}") #)
```

###### in the future:
```
ltrim: {-varnme}
rtrim: {varname-}
trim: {-varname-}
trim: {trim:varname}
trim: {$varname:trim}
```
### variable modifier for var_export
###### in the present:
```
// create subparser
function export($src, $items){
    $v = $items[$src];
    if (is_object($v))
    $v = get_object_vars($v);
    return str_replace("\n","",var_export($v, true));
}

div::setSubParser('export');
```

```
{export}varname{/export}
```

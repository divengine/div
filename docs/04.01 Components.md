# 4.1 Components

Components are reusable template fragments that encapsulate layout logic and can be composed into larger views.

## 8.1.1 Build a component

combobox.tpl

```
<select id="{$id}" name="{$name}">
    [${$options}]
        <option value="{$val}">{$text}</option>
    [/${$options}]
</select>
```

This component uses [[02.03 Simple replacements]], [[02.10 Lists (loops)]], and [[04.02 Recursion]].

## 8.1.2 Use a component

index.tpl

```
[[_empty
    {= id: "products" =}
    {= name: "products" =}
    {= options: "products" =}
    {% combobox %}
_empty]]
```

Using a component is equivalent to including a template fragment, often inside a capsule.

## 8.1.3 Provide data

index.php

```php
<?php

echo new div('index.tpl', [
    'products' => [
        ['val' => 1, 'text' => 'Banana'],
        ['val' => 2, 'text' => 'Potato'],
        ['val' => 3, 'text' => 'Apple']
    ]
]);
```

Output

```
<select id="products" name="products">
    <option value="1">Banana</option>
    <option value="2">Potato</option>
    <option value="3">Apple</option>
</select>
```

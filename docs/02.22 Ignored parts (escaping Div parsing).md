# 2.22 Ignored parts (escaping Div parsing)

Ignored blocks are passed through unchanged. The parser does not interpret tags inside these blocks.

**Syntax**

```
{ignore}
    ... some ignored code here ...
{/ignore}
```

**Example**

index.php

```php
echo new div('index.tpl', ['name' => "Peter"]);
```

index.tpl

```
{ignore}

Name: {$name}

{/ignore}
```

Output

```
Name: {$name}
```

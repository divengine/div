# 2.23 Strip or clean the resulting code

Strip blocks normalize whitespace in the output by removing redundant spaces and newlines.

**Syntax**

```
{strip}
    ... some code here ...
{/strip}
```

**Example**

index.tpl

```
{strip}
Hello Jack, ...

          ...the previous lines are of more.
{/strip}
```

Output

```
Hello Jack, ...
...the previous lines are of more.
```

# 2.39 Dialect translator

Div can translate from any dialect to the current dialect. Translation uses the current template variables for context, so you must create a div instance before translating.

**Example**

index.php

```php
<?php

include 'div.php';

$tpl = new div('index.tpl');

$tpl->translateFrom([
    'DIV_TAG_IGNORE_BEGIN' => '{literal}',
    'DIV_TAG_IGNORE_END' => '{/literal}'
]);

$tpl->show();
```

index.tpl

```
{= name: "Peter" =}

{literal}
    {$name}
{/literal}

{$name}
```

index.tpl (translated)

```
{= name: "Peter" =}

{ignore}
    {$name}
{/ignore}

{$name}
```

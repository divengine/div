# 2.40 Custom modifiers

Custom modifiers extend the variable modifier system. A custom modifier can be a function or a static method, and it must accept a single parameter.

**Example**

index.php

```php
<?php

include 'div.php';

class MyModifiers {
    public static function upper($value){
        return strtoupper($value);
    }
}

function lower($value){
    return strtolower($value);
}

div::addCustomModifier('upper:', 'MyModifiers::upper');
div::addCustomModifier('lower:', 'lower');

echo new div('index.tpl', ['text' => 'Hello World']);
```

index.tpl

```
{upper:text}

{lower:text}
```

Output

```
HELLO WORLD

hello world
```

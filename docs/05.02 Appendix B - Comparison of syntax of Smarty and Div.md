# 5.2 Appendix B - Comparison of syntax of Smarty and Div

This appendix compares equivalent syntax in Smarty and Div.

## 11.2.1 Loops

Smarty:

```html
{foreach $foo as $bar}
      <a href="{$bar.zig}">{$bar.zag}</a>
      <a href="{$bar.zig2}">{$bar.zag2}</a>
      <a href="{$bar.zig3}">{$bar.zag3}</a>
    {foreachelse}
      There were no rows found
{/foreach}
```

Div:

```html
[$foo]
  <a href="{$zig}">{$zag}</a>
  <a href="{$zig2}">{$zag2}</a>
  <a href="{$zig3}">{$zag3}</a>
@empty@
  There were no rows found
[/$foo]
```

## 11.2.2 Include

Smarty:

```html
{include file="header.tpl"}
```

Div:

```html
{% header %}
```

## 11.2.3 Iterations

Smarty:

```html
{for $x = 1 to 20 step 2}
      {$x}
{/for}
```

Div:

```html
[:1,20,x,2:]
      {$x}
[/]
```

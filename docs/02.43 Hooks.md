# 2.43 Hooks

Hooks are methods you can implement in a class that extends div. They are invoked at specific points in the lifecycle.

Available hooks:

- `beforeBuild`
- `afterBuild`
- `beforeParse`
- `afterParse`

The `beforeBuild` hook can modify `$src` and `$items` before parsing begins.

**Example**

index.php

```php
class Page extends div{
    public function beforeBuild(&$src = null, &$items = null){
        $this->title = 'Hello World';
        $items['body'] = 'This is the hook!';
    }
}

echo new Page('index.tpl');
```

index.tpl

```
<h1>{$title}</h1>
<p>{$body}</p>
```

Output

```
<h1>Hello World</h1>
<p>This is the hook!</p>
```

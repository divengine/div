# 2.37 Dialects

Dialects allow you to redefine template tags by overriding constants that start with `DIV_TAG`. This lets you map Div syntax to another template language or avoid collisions with existing markup.

Define dialect constants before loading `div.php`. You only need to override the constants you want to change; the rest use defaults.

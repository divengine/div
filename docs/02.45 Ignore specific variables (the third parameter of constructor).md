# 2.45 Ignore specific variables (the third parameter of constructor)

You can tell the engine to ignore certain variables by passing a list of names as the third constructor parameter.

```php
/* Third parameter as array */

echo new div('index.tpl', ['name' => 'Peter'], ['name']);

/* Third parameter as string */

echo new div('index.tpl', ['name' => 'Peter', 'age' => 25, 'sex' => 'M'], 'name,age');
```

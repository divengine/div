# 2.10 Lists (loops)

Lists repeat a block of template code for each item in a collection.

**Syntax**

```
[$listvar]
    ... some code here ...
@empty@
    ... some code when the list is empty ...
[/$listvar]
```

## 6.10.1 Breaking a loop

You can stop iteration using `@break@`. Anything after the break tag in the loop body is ignored and the loop stops.

Example

```
[$products]
    {?( {$_index} == 3 )?} <hr> @break@ {/?}
    {$value}<br>
[/$products]
```

**Example**

```php
echo new div('index.tpl', [
    'employees' => ['Rafa', 'Peter', 'John'],
    'products' => []
]);
```

**index.tpl**

```
Employees:

[$employees]
    {$value}
[/$employees]

Products:

[$products]
    {$name}
@empty@
    Empty list of products!
[/$employees]
```

**Output**

```
Employees:

Rafa
Peter
John

Products:

Empty list of products!
```

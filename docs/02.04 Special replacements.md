# 2.4 Special replacements

Special replacements emit control characters that are otherwise difficult to write in templates.

|Tag|Replacement|
|---|---|
|{\n}|\n|
|{\r}|\r|
|{\t}|\t|
|{\v}|\v|
|{\f}|\f|
|{\$}|$|

**Example**

index.tpl

```html
Hello{\n}Peter
{\t}Today is {/div.now:Y-m-d/}
```

Output

```html
Hello
Peter
        Today is 2013-07-24
```

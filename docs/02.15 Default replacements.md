# 2.15 Default replacements

Default replacements map specific input values to alternative output values.

## 6.15.1 PHP usage

```php
<?php

div::setDefault($value_to_search, $replace_with);
```

## 6.15.2 Template usage

```
{@ [value_to_search, replace_with] @}
```

## 6.15.3 Example

index.php

```php
<?php

div::setDefault(true, "YES");

echo new div("index.tpl", [
    "haveproducts" => true,
    "havemoney" => false
]);
```

index.tpl

```
{@ [false, "NO"] @}

Have products: {$haveproducts}
Have money: {$havemoney}
```

Output

```
Have products: YES
Have money: NO
```

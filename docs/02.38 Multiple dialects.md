# 2.38 Multiple dialects

You can set a dialect per template using the `@_DIALECT` property. The dialect file contains a JSON object of `DIV_TAG_*` overrides.

**Example**

index.tpl

```
@_DIALECT = smarty.dialect
{* this is a comment *}
Name: {$name}
{literal}
{$name}
{/literal}
{% other %}
```

other.tpl

```
@_DIALECT = twig.dialect
{{ foo.bar }}
```

smarty.dialect

```
{
  'DIV_TAG_IGNORE_BEGIN': '{literal}',
  'DIV_TAG_IGNORE_END': '{/literal}',
  'DIV_TAG_COMMENT_BEGIN': '{*',
  'DIV_TAG_COMMENT_END': '*}'
}
```

twig.dialect

```
{
  'DIV_TAG_REPLACEMENT_SUFFIX': ' }}',
  'DIV_TAG_MODIFIER_SIMPLE': '{ '
}
```

index.php

```php
<?php

include "div.php";

echo new div("index.tpl", [
    'name' => 'Peter',
    'foo' => [
        'bar' => 45
    ]
]);
```

Output

```
Name: Peter

{$name}

45
```
